# Loads

## Gravity Load

Applies a gravitational body force to the entire model.

| Parameter  | Description                    |
| ---------- | ------------------------------ |
| Direction  | Gravity vector (e.g. 0, 0, -1)|
| Magnitude  | Acceleration (e.g. 9.81)      |

## Nodal Load

Applies a concentrated force or moment at selected nodes.

| Parameter  | Description                    |
| ---------- | ------------------------------ |
| Node       | Target node(s)                 |
| Force      | Force vector (Fx, Fy, Fz)     |
