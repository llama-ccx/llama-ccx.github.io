# Elements

Lama supports the following CalculiX element types:

## Shell Elements

| Element  | Nodes | Description                   |
| -------- | ----- | ----------------------------- |
| Shell3   | 3     | Triangular shell              |
| Shell4   | 4     | Quadrilateral shell           |
| Shell6   | 6     | Quadratic triangular shell    |
| Shell8   | 8     | Quadratic quadrilateral shell |

## Solid Elements

| Element  | Nodes | Description                        |
| -------- | ----- | ---------------------------------- |
| Tetra4   | 4     | Linear tetrahedron                 |
| Tetra10  | 10    | Quadratic tetrahedron              |
| Hexa20   | 20    | Quadratic hexahedron (brick)       |
