# Analysis Steps

Steps define the type of analysis to perform.

## Linear Static

A single-step linear elastic analysis. The most common step type for basic structural problems.

## Nonlinear Static

An incremental nonlinear static analysis with load stepping and convergence control.

## Dynamic Implicit

A time-history dynamic analysis using implicit time integration.

## Frequency

A modal/eigenvalue analysis to extract natural frequencies and mode shapes.

## Output Requests

Each step can include output requests for nodal and element results (displacements, stresses, reaction forces, etc.).
