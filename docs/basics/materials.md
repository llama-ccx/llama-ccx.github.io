# Materials

Llama supports several material models for use with CalculiX.

## Isotropic Material

A linear elastic isotropic material defined by:

| Parameter        | Description                  |
| ---------------- | ---------------------------- |
| Young's Modulus  | Elastic modulus (E)          |
| Poisson's Ratio  | Poisson's ratio (ν)         |
| Density          | Mass density (ρ)             |

## Orthotropic Material

An orthotropic material with independent properties in three directions.

## Spring Material

A spring-type material for point-to-point spring elements.

## Stiffness Matrix Material

A general stiffness matrix material for advanced element formulations.

!!! note
    For most structural analyses, the **Isotropic Material** component is sufficient.
