# Analysis

## Build Input Deck

The **Build Input Deck** component writes the assembled model to a CalculiX `.inp` file.

## Run Analysis

The **Run with Exe** component executes CalculiX on the generated input deck.

| Parameter       | Description                         |
| --------------- | ----------------------------------- |
| Input Deck Path | Path to the `.inp` file             |
| CalculiX Path   | Path to the `ccx` executable        |

!!! note
    Ensure CalculiX is installed and the path is correct before running.
